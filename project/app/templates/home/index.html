The Otherwise Summary Maker, based on the Test Driven Development FastAPI course (using Docker)

You can use this API to summarise a web page. Here is an example of using httpie to POST a url:

  • http --json POST https://summary-maker.herokuapp.com/summaries/ url=https://www.idjotherwise.github.io/nlp-otherwise

    • Or using curl

  • curl -d '{"url":"https://idjotherwise.github.io/nlp-otherwise"}' -H "Content-Type: application/json" -X POST https://summary-maker.herokuapp.com/summaries/
